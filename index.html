The Web Share API allows you to share text, URLs, or files. In its simplest form, the share code looks something like this: try { await navigator.share({ title: 'Title', text: 'Text', url: location.href, }); } catch (err) { console.error(`${err.name}: ${err.message}`); } If a sharing action needs to happen in a third-party iframe, a recent spec change requires you to explicitly allow the operation. Do this by adding an allow attribute to the tag with a value of web-share. This tells the browser that the embedding site allows the embedded third-party iframe to trigger the share action. Web Share in third-party iframes You can see this in action in a demo on Glitch and view the source code. Failing to provide the attribute will result in a NotAllowedError with the message Failed to execute 'share' on 'Navigator': Permission denied.
